      What the Model Detects

      Trained on 1,091 real mulberry leaf images from Bangladesh sericulture farms

      Healthy

      Disease-free leaf with uniform green color, smooth texture and no visible spots or discoloration.

      440 training images
      No treatment needed

      Leaf Rust

      Irregular reddish-brown pustules on leaf surface. Caused by Cerotelium fici fungal pathogen.

      489 training images
      Fungicide treatment needed

      Leaf Spot

      Dark brown circular spots with yellow halos. Caused by Cercospora moricola fungus.

      162 training images
      Copper fungicide needed

      Explainable Deep Learning for Automatic Mulberry Leaf Disease Classification

      Source of the dataset used in this project. PDS-CNN model trained on real mulberry leaf images (Leaf Rust + Leaf Spot + Healthy) collected from Bangladesh sericulture farms. Uses Grad-CAM for explainability.

      CNN-based classification model for mulberry leaf disease recognition. Same dataset authors. Outperforms existing SOTA approaches in disease identification accuracy.

      Dataset · Kaggle

      Mulberry Leaf Dataset — Nahiduzzaman et al.

      1,091 real mulberry leaf images captured with DSLR camera from farms in Rajshahi, Bangladesh. 3 classes: Disease Free (440), Leaf Rust (489), Leaf Spot (162). Annotated by sericulture experts.
